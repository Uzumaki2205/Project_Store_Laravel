## SETUP

-   Copy file .env.example thành file .env
-   Đổi config database

```bash
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=<tên database>
DB_USERNAME=root
DB_PASSWORD=
```

-   Đổi config mail

```bash
MAIL_MAILER=smtp
MAIL_HOST=smtp.gmail.com
MAIL_PORT=587
MAIL_USERNAME=<username>
MAIL_PASSWORD=<password>
MAIL_ENCRYPTION=tls
```
